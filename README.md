# Student Management System

A Python-based student record management system using CSV files and Tkinter GUI.

## Features
- Add Student
- View all Students
- Search Student by Roll Number
- Update Student Details
- Delete Student
- GUI Version Included (Tkinter)
- CSV File Storage (students.csv)
